Este projeto é uma Calculadora de IMC (Índice de Massa Corporal) feita usando HTML, CSS e JavaScript.
Eu desenvolvi acompanhando um vídeo no YouTube, onde aprendi a estruturar a página, estilizar e adicionar lógica com JavaScript.

No HTML, criei a estrutura da página com campos para nome, altura e peso, um botão para calcular e uma área para mostrar o resultado.
No CSS, estilizei a calculadora com um tema escuro, deixando o layout centralizado, organizado e mais bonito de usar.
No JavaScript, aprendi a capturar os valores digitados pelo usuário, calcular o IMC usando a fórmula correta e mostrar o resultado na tela juntamente com a classificação (abaixo do peso, peso ideal, obesidade, etc).

Quando o botão “Calcular” é clicado, o programa verifica se todos os campos estão preenchidos. Se estiverem, ele faz o cálculo e mostra o resultado. Caso contrário, exibe uma mensagem pedindo para completar os campos.

Esse código me ajudou a entender melhor como HTML, CSS e JavaScript trabalham juntos, principalmente o uso de eventos, manipulação do DOM e validação básica de dados.
